## ğŸš€ Quick Start

### Prerequisites
- Python 3.9 or higher
- Git installed
- API keys (Groq, OpenAI, etc.)
- Basic Python knowledge

### Clone the Repository

```bash
git clone https://github.com/ms-afridi/top-30-llm-projects.git
cd top-30-llm-projects
```

### Navigate to Any Project

```bash
cd day-1-llm-cli-chatbot
```

### Follow Project-Specific Instructions

Each project has its own README with detailed setup instructions:

1. Install dependencies: `pip install -r requirements.txt`
2. Set up environment variables (`.env` file)
3. Run the application: `python app.py` or `streamlit run app.py`
